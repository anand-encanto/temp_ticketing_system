TICKETING SYSTEM OVERVIEW

1. What is this application?
   The Ticketing System is a simple tool designed to help you report issues and track their progress until they are resolved. It ensures that every problem is assigned to the right person and handled efficiently.

2. Key Features

- Create tickets easily to report any problem or request.
- Track the status of your reports in real-time.
- Communicate directly with support staff through comments.
- Clear organization by location and department.

3. How to use the system

3.1 Accessing the System

1. Open the application in your web browser.
2. Log in using your registered email and password.
3. If you do not have an account, click on the registration link to create one.

3.2 For General Users (Reporting Issues)

- Creating a Ticket: Select the "Create Ticket" option, choose the relevant department and location, and provide a clear description of the issue.
- Tracking Progress: Go to the "My Tickets" section to see all your submitted reports and their current status (such as Open, In Progress, or Resolved).
- Adding Information: You can click on any ticket to add more comments or read updates from the staff.

    3.3 For Support Staff (Executives)

- Viewing Tasks: Check the "Assigned List" to see all tickets assigned to you for resolution.
- Updating Status: When you begin work, update the ticket status to "In Progress." Once the issue is fixed, mark it as "Resolved."
- Communication: Use the comment feature to ask the user for clarification or provide updates on your progress.

    3.4 For Administrators (System Management)

- Managing Users: Use the "User Management" tools to add new staff members or edit existing user profiles.
- System Setup: Manage the list of available "Departments" and "Locations" to keep the system organized.
- Performance Overview: Access the dashboard to view summaries of ticket activity and response times.
